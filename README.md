Projeto relativamente simples para criptografar uma mensagem via command line na file encrypt que ira gerar o codigo(key) , o usuario vai digitar sua frase/palavra 
e ira gerar um ciphertext , somente apos isso no arquivo decrip iremos colocar a key gerada no encrypt e tambem colocar o ciphertext para decifrar a nossa frase/palavra
Bibliotecas usadas:
  cryptography
  getpass
